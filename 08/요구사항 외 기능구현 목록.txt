1. 스크롤 구현 ( mouseon 부분 )
	- 스크롤을 조정해줄 필요가 없는 경우를 제외하고 ( 예를 들어, 모든 내용이 화면 하나에 들어갈 경우 )
              트렌딩, 이슈, 엔터에 마우스를 올릴경우 해당 Section 으로 스크롤 이동 ( active 부분도 해결 )

2. 스크롤 구현 
	- 사용자가 스크롤을 이동 시킬 경우 구역에 맞게 active 되는 메뉴를 바꿔준다.

3. 1boon 메뉴 클릭시 홈으로 이동

4. menu 창을 sticky 로 설정하여 언제든 클릭이 가능하도록 하였다.

5. loading 중 아이콘을 spin 되도록하였다.

6. 'More ..' 버튼을 눌렀을 때 창이 자동적으로 내려간다.

7. ★ 만약 다음 페이지 정보 (hasNext) 가 없다면 메뉴 버튼을 숨긴다.

8. 기사제목을 사진에 녹였다.
